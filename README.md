README
======

BackboneJS marionette POC

Requirements
------------

	node.js >= 0.8
	grunt-cli
	npm
	stylus
	bower

Installation
------------

	bower install
	npm install
	grunt run

Build production apps
---------------------

	grunt production

Ready to use production application can be found in /build folder.
